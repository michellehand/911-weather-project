# Project Scope

## Objective

Determine whether weather conditions can help predict increases in 911 emergency call volume.

The project aims to build a predictive analytics pipeline that combines:
- Historical emergency call data
- Historical weather data
- Time-based trends and seasonality

The final output will include predictive models, visualizations, and an interactive dashboard.

---

## Core Question

Can weather patterns improve prediction accuracy for 911 emergency call demand?

---

## Initial Scope

### Included
- Historical 911 call analysis
- Weather data integration
- Exploratory data analysis
- Feature engineering
- Predictive modeling
- Dashboard creation
- Portfolio-ready documentation

### Excluded (for MVP)
- Real-time streaming systems
- Deep learning models
- Live emergency dispatch integration
- Mobile application development
- Large-scale cloud infrastructure
